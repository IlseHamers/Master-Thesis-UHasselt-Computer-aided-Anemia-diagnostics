# Master-Thesis-UHasselt-Computer-aided-Anemia-diagnostics
Scripts used in the UHasselt master thesis of Ilse Hamers on Computer-aided Anemia Diagnostics at the MUMC+. Scripts are divided into data cleaning and preparation, prompting, and analysis. Note that patient data cannot be shared and is not available in this repository. Therefore, sample IDs were taken out of the code.

All model prompting was done on a NVIDIA GeForce RTX 5090 (32 GB VRAM).

Note that not all scripts are originally written by the student. The scripts for prompting and QLoRA are edits on existing MUMC+ scripts.
